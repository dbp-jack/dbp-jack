- 팀 구성: 3명 / 역할: 팀장  
- 담당 : 백엔드 개발, 시스템 설계 및 구축, 논문 작성 및 교류회 발표
- 구현/성과:  
  - 초기에는 setInterval 기반 AJAX 방식으로 실시간 ECG 데이터를 처리했으나, 서버 부하 증가와 데이터 지연·누락 문제가 발생
